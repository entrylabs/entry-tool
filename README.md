## Entry Tool

이 Repository 는 [EntryJS](https://github.com/entrylabs/entryjs)에서 사용되는 팝업등의 다양한 기능들을 위해 만들어 졌습니다.

:tada:

## 설치

```bash
yarn add entry-tool
```

## 개발

소스 다운로드

```bash
git clone https://github.com/entrylabs/entry-tool
```

entry-tool 실행

```bash
yarn
yarn start
#or
npm install
npm run start
```
